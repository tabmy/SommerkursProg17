Oppgavesett 8 - Nettsider: HTML, CSS og Javascript

Bruk gjerne internet for å finne svar på oppgaver

1) HTML og CSS

  • Hva står CSS for og hva brukes det til? Hvorfor kaller vi CSS et presentasjons s pråk/markerings språk?
  • Nevn noen eksempler på hvilke egenskaper CSS har til å påvirke designet på en nettside.
  • Bruk internett og skriv ned hvordan man endrer følgende egenskaper i CSS: bakgrunnsfarge, fonttype, plass over et element og fargen på en lenke.
  • Hva må vi gjøre for å linke dokumentene når vi har CSS-kode i en annen fil enn HTML-dokumentet?

2) Javascript

  • Beskriv hva vi hovedsakelig bruker JavaScript til på nettsider. Hvorfor er JavaScript nyttig sammen med HTML og/eller CSS?
  • Hvordan definerer man og tilegner en variabel verdi i JavaScript?
  • Hvordan ser de typiske kontrollstrukturene ut i JavaScript ut (if, for, while...)?
  • Hvor skal vi plassere JavaScript kode? Hvilke andre muligheter har man, og hvilken av mulighetene er oftest brukt?

3) lag din egen hjemmeside!

I denne oppgaven skal vi starte med å lage en liten hjemmeside. Dette foregår steg for sted, og bruk gjerne internett for å finne svar om du ikke finner ut av noe. Vi skal også fortsette å bruke denne hjemmesiden i noen av oppgavene i morgen.

  • Begynn med å åpne en teksteditor(notepad(Windows) eller TextEdit(Mac)), eller last ned et program som Brackets. Fyll inn det enkle HTML-eksempelet fra modul 8.
