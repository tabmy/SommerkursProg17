# SommerkursProg17

Forberedelseskurset i programmering gir en innføring i grunnleggende datamaskinkunnskap og programmeringsteori som fører til bedre forståelse av og kunnskapsgrunnlag til kursene i første semester til datalinjene (Dataingeniør, Anvendt data og Informasjonsteknologi).

 

Kurset vil gå gjennom generell studieteknikk og studietips, datamaskinkomponenter og binærtall, programmeringskonsepter og -språk, i tillegg til grunnleggende nettsideutvikling. Kurset er i hovedsak praktisk og vil bestå bl.a. av oppgaveløsning og workshops.
