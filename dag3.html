Intro til programmering i Java og PHP

Variabler

Som nevnt tidligere er programmeringsspråk en måte å fortelle datamaskinen hva som må gjøres for å komme frem til et ønsket resultat. Vi kan si at en oppskrift for å lage en brødskive krever at du skal ha:

  • 1 stk brød
  • 7.6 gram smør
  • 2 skiver ost
  • Ikke skinke

Altså, vi trenger å lagre disse verdiene når vi skriver algoritmen vår for å lage en brødskive, her kommer da konseptet 'variabler' inn. En variabel kan lagre data av typene heltall, desimaltall, tegn og tegnstrenger og bolske verdier (sann/falsk). Det er litt forskjellig hvordan man representerer en variabel i forskjellige programmeringsspråk. Vi skal først bruke programmeringsspråket Java for å se hva slags variabler man har der:

Variabler i java

Heltallstyper

I Java representerer man heltall ved 4 forskjellige typer: (navn, størrelse)

  • byte, 1 byte
  • short, 2 byte
  • int, 4 byte
  • long, 8 byte

Som du ser har disse forskjellige størrelser, talltypen 'byte' har størrelsen 1 byte, som vil si at den kan holde på 2^8 = 256 tall. Siden heltallstyper også skal kunne holde på negative tall, er den minste verdien til en byte -127, og den maksimale verdien 128 (den må også holde på tallet 0).

Desimaltallstyper

Desimaltall er mer anvendelige i situasjoner hvor man f.eks. i et banksystem hvor man ønsker å holde oversikt over saldo og transaksjoner. I Java har man to desimaltallstyper:

  • double
  • float

Tegn-verdier

Man trenger også variabler for å holde på tekst, som i java er:

  • char
  • String

En 'char' er en representasjon av én bokstav, mens en 'String' er en samling av flere bokstaver, altså en samling av flere 'char'. String brukes da naturligvis for å representere tekst.

Boolske verdier

En boolsk variabel er en variabel som kan ha to verdier. I Java kalles denne typen 'boolean', og har verdiene 'true' eller 'false'.

Deklarasjon av variabler i Java

Å deklarere en variabel betyr å lage en variabel når man programmerer - og gi den en verdi. Det er forskjellig hvordan man deklarerer en variabel, avhengig av typen til variabelen. Generelt så deklarerer man variabler med å skrive '<type> <navn> = <verdi>;'. Her er noen eksempler:

int alder = 23; double vekt = 90.4; char førsteBokstav = 'A'; String navn = "Andreas"; boolean erStudent = true;

Ser du nøye på koden over, så er det flere ting man må legge merke til. Det første er semikolon, som settes etter hver instruksjon man skriver, for å fortelle datamaskinen at dette er slutten av instruksjonen. Når man deklarerer en 'char' variabel, så må verdien dens settes mellom to enkle 'hermetegn', mens verdien til en String variabel settes mellom to doble "hermetegn".

Etter man har deklarert en variabel (gitt den navn, type og evt verdi), så kan man endre på variabelen sin verdi på denne måten:

int a = 2; int b; b = a + 3;

Talltyper støtter også flere operasjoner som å addere (+), subtrahere (-), dividere (/), multiplisere (*) og modulo (%). Man kan også utføre '+' operasjoner med strenger (String variabler), og gjøres slik:

String fornavn = "Andreas"; String etternavn = "Strand"; String navn = fornavn + " " + etternavn; //navn har nå verdien "Andreas Strand"

Legg også merke til det som står etter '//' i forrige linje. Alt man skriver etter '//' blir ikke lest som kode av datamaskinen, men gir programmerer mulighet til å kommentere koden sin.
